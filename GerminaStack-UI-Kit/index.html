Visao geral

O kit foi extraido do prototipo e reorganizado em fundacoes reutilizaveis: tokens, layout, navegacao, cards, formularios, feedback visual e um conjunto pequeno de comportamentos acessiveis ativados via atributos data-gs-*.

VisualAzul-marinho, laranja, superficies claras e cantos arredondados.
ComportamentoTabs, menus, toasts, dismiss e interacoes simples sem framework.
EntregaUso direto no navegador, com HTML declarativo e baixa friccao para novos devs.

Instalacao

Inclua os arquivos abaixo e comece a montar a tela com as classes do kit.

Bootstrap minimo 
<link rel="stylesheet" href="./assets/css/germinastack.css" />
<script src="./assets/js/germinastack.js" defer></script>

<main class="gs-page">
  <section class="gs-card">Seu conteudo</section>
</main>

Tokens

Os tokens ficam em :root. A regra para customizacao e sobrescrever somente variaveis, sem alterar classes base.

--gs-navy --gs-orange --gs-surface-soft
Override recomendado 
:root {
  --gs-navy: #00284f;
  --gs-orange: #ffba52;
  --gs-page: #f0f2f5;
}

Materias em alta

JavaScript

O arquivo JS expone uma API pequena e previsivel em window.GerminaStackUI.

  • GerminaStackUI.init(root): inicializa tabs, menus, toasts, dismiss e demos.
  • GerminaStackUI.showToast({ title, message, tone }): cria toast programaticamente.
  • data-gs-tabs: agrupa tabs e paines.
  • data-gs-menu: agrupa trigger e painel de menu.
  • data-gs-dismiss: esconde banners e alerts.
  • data-gs-copy: copia blocos de codigo da documentacao.
Toast manual 
window.GerminaStackUI.showToast({
  title: "Deploy concluido",
  message: "A pagina de perfil foi atualizada com o kit.",
  tone: "success"
});

Boas praticas

Estas regras evitam degradacao visual e mantem a coerencia com o prototipo.

  • Sempre prefira sobrescrever tokens antes de criar novas cores soltas.
  • Use gs-card, gs-post e gs-side-card como blocos de composicao.
  • Mantenha botoes primarios reservados para acao principal da area.
  • Para novas interacoes, reaproveite o padrao data-gs-* em vez de amarrar seletores fragis.
  • Em mobile, reduza quantidade de colunas antes de mexer em espacamento e tipografia.
