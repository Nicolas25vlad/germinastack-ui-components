# GerminaStack Componentes

Kit de componentes em HTML, CSS e JavaScript vanilla derivado do `GerminaStack-Prototype`.

## Estrutura

- `GerminaStack-Prototype/`: prototipo original usado como referencia visual e funcional
- `GerminaStack-UI-Kit/`: kit reutilizavel, documentacao e playground

## Como usar

Abra `GerminaStack-UI-Kit/index.html` no navegador para ler a documentacao.

Abra `GerminaStack-UI-Kit/playground.html` para ver uma composicao completa dos componentes.

## Entrega

O kit foi organizado para uso sem build, plugins ou dependencias externas:

- `assets/css/germinastack.css`: tokens, base, layout e componentes
- `assets/js/germinastack.js`: comportamentos reutilizaveis por `data-*`
- `assets/js/docs.js`: recursos da documentacao

## Versionamento sugerido

O historico git desta entrega foi separado em etapas de fundacao, documentacao e acabamento para facilitar manutencao futura.
